# Integration Standards: ERP Connectors

## 1. SAP S/4HANA (IDoc)
Signals generated by the Scout are transformed into **SAP IDoc (Intermediate Document)** format for ingestion.

### Mapping Profile: `PREQCR1` (Purchase Requisition Create)
| Scout Field | IDoc Segment | Field Name | Transformation |
|-------------|--------------|------------|----------------|
| `signal_id` | `E1PREQCR1` | `BANFN` | Hashed UUID (truncated 10 chars) |
| `timestamp` | `E1PREQCR1` | `BLDAT` | YYYYMMDD Format |
| `title` | `E1BPMEPOHEADER`| `SHORT_TEXT`| Truncate to 40 chars |
| `sap_bapi_transaction_code` | `E1BPMEPOTEXT` | `TEXT_ID` | Maps to `RELEASE_BLOCK_Z1` or `UPDATE_SS_Z2` |

## 2. Oracle Cloud SCM (REST)
For Oracle environments, we utilize the **Oracle Fusion Cloud SCM REST API**.
- Endpoint: `/fscmRestApi/resources/11.13.18.05/purchaseRequisitions`
- Authentication: OAuth 2.0 (Client Credentials Flow)

## 3. Middleware
Integration is decoupled via an Enterprise Service Bus (MuleSoft/Apache Camel) to ensure failure isolation.
