# Glossary of Terms

| Term | Definition |
|------|------------|
| **Solicitation** | A procurement notice from a government entity requesting bids/proposals. |
| **Signal** | A standardized JSON object representing a vetted procurement opportunity. |
| **Demand Probability** | A computed float [0.1-1.0] indicating the likelihood a document matches the user's intent. |
| **Readiness Protocol** | The overarching algorithmic logic that switches procurement from "Just-in-Time" to "Just-in-Case." |
| **Resilience Premium** | The calculated cost (holding cost) required to maintain inventory buffers for critical assets. |
| **Criticality Threshold** | The financial tipping point (found to be ~$800k in simulations) where the cost of failure exceeds the cost of readiness. |
| **CUI** | Controlled Unclassified Information (Requires NIST 800-171 compliance). |
